The add-on will now download channel logos and cache them locally. This may take about 1 or 2 minutes.

You only need to do this once.


The following options are available in the add-on Settings:

Sort channels alphabetically
Remember last genre filter
Choose fullscreen or panel display
Enable add-on specific debug logging
Disable use of ISA for playback
EPG cache duration
Clear EPG cache and thumbs


The following options are available from the context menu:

Program description
Channel description
Filter channels by category
Clear category filter (only available when a category filter is applied)
Add Channel to FAVORITES (not available while viewing Favorites)
View Favorites (only available if Favorites exist)
Remove Channel from favorites (only available while viewing Favorites)
Clear Favorites (only available while viewing Favorites)
Exit Favorites (only available while viewing Favorites)
